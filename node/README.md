Node.js Automation Layer for AE Workflow
Опис
Ця папка містить інструменти для автоматизованого запуску скриптів After Effects (ExtendScript/JSX) через Node.js.
Node.js-оболонка дозволяє виконувати повний цикл автоматизації без ручного втручання — забезпечує послідовний запуск, контроль логування та перевірку правильності доступу до проєкту та матеріалів.

Основний скрипт
index.js
Менеджер автоматичного запуску всіх необхідних JSX-скриптів. Додає зручний контроль послідовності, повідомлення про помилки, інтеграцію логів прямо в консоль.

Запуск
Встановіть Node.js (https://nodejs.org/)

Всі шляхи та параметри проєкту налаштуйте у ../js/config.json.

Відкрийте термінал/консоль у цій папці.

Запустіть команду:

text
node index.js
After Effects автоматично відкриє потрібний .aep, внесе зміни згідно задачі, збереже логи та фінальний результат.

Додаткові відомості
AE_PATH / afterfx
Для запуску скриптів використовується консольна команда After Effects (afterfx.exe -r script.jsx).
Якщо шлях до afterfx.exe нестандартний, вкажіть змінну середовища AFTERFX_PATH.

Логування
Усі проміжні логи автоматично зберігаються у ../logs/. Помилки та результат запуску відображаються у консолі.

Рекомендації

Запускайте із закритим After Effects — оболонка сама відкриє і обробить потрібний проект згідно маршруту з config.json.

Перед запуском переконайтеся, що всі файли й каталоги згідно структури наявні (особливо .aep та відео).
